# BinaryNinja Plugin
This is the BinaryNinja interface to DeepReflect.

Thank you to [Scott Bergstresser](https://github.com/sab4tg) for creating the alpha version of this plugin.

## Setup
  - Requirements:
    - BinaryNinja 2.3
    - psycopg2
  - Setup:
    ```
    $ cp -r ./binaryninja/deepreflect ~/.binaryninja/plugins/
    ```
